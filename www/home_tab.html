Welcome to TOTEM web tool

  • TOTEM is a web tool designed to provide spatio-temporal information on gene lists, contributing to their functional characterization with single-cell resolution.
  • It calculates enrichment values per tissue based on a user-provided gene list and a reference gene expression atlas.
  • TOTEM depicts enrichment values in a diagram of the chosen organism and identifies tissue-specific genes from user's list.
  • Available organisms are: Arabidopsis thaliana, Sorghum bicolor and Solanum lycopersicum
